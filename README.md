# AI Powered Resume Maker

## Overview
AI Powered Resume Maker is a web application that helps users create, analyze, and rank resumes using Artificial Intelligence, NLP, and Machine Learning techniques.

## Features
- Resume Upload
- Resume Parsing
- Skill Extraction
- Resume Ranking
- Flask Web Interface
- PDF Resume Support

## Technologies Used
- Python
- Flask
- Machine Learning
- NLP
- Scikit-learn
- HTML
- CSS

## Project Structure

```bash
AI-Powered-Resume-Maker/
│
├── static/
├── templates/
├── uploads/
├── app.py
├── requirements.txt
└── README.md
```

## How to Run the Project

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

## Output
The system analyzes resumes and ranks them based on skills and relevance.
